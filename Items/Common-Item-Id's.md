**Note:**

Axes degrade twice as fast when used as weapons. Use DURABILITY
enchantment to counteract or balance with more damage, etc...

Stick, blaze rod and arrow are good options for blunt, staff, and dagger
type weapons with no durability loss but you must use the Damage option
to make them do damage.

[1] These values are based the attributes you have to use when creating
an item. See item attributes page.
