**Description:** If the target is within the given WorldGuard region.

---

**Attributes:**

| Attribute | Aliases        | Description               |
| --------- | -------------  | ------------------------- |
| region| r, name, n | The region to check|

---

**Examples:**

```
  Conditions:
  - region{r=someregion} true
```
