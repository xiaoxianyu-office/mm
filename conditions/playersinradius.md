**Description:** Checks how many players are in a radius.

---

**Attributes:**

| Attribute | Aliases        | Description               |
| --------- | -------------  | ------------------------- |
| amount | a | the given range value to check | >0 |
| radius | range, r | the given radius to check | 32 |

Example
------

```
Conditions:
- playersinradius{a=>3;r=16}
```
