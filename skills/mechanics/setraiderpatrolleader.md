Mechanic: setRaiderPatrolLeader (5.1.4)+
======================
Sets the raider patrol leader

Attributes
----------

| Attribute | Aliases | Description                            | Default |
|-----------|---------|----------------------------------------|---------|
|  leader   |         |   sets leader true or false            |   true  |

  

Examples
--------

      - setRaiderPatrolLeader{leader=true} @target
