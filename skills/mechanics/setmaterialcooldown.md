## Description
Sets a cooldown on items


## Attributes

| Attribute    | Aliases        | Description                                                                               | Default |
|--------------|----------------|-------------------------------------------------------------------------------------------|---------|
| Material         | m              | The material                           | ENDER_PEARL |
| Duration         | d              | The cooldown duration                           | 100 |


## Examples
```yaml
  Skills:
  - setmaterialcooldown{m=CHORUS_FRUIT;d=150} @self
```
