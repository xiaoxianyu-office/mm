## Description
Applies a bone meal effect to the target blocks.


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| blockFace | bf, face, f | The [block face](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/block/BlockFace.html) to apply bonemeal to | UP<!--type:BlockFace--> |


## Examples
```yaml
  Skills:
  - bonemeal @selflocation
```
