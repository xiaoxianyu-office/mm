# Description
Checks for tags set in the [Damage](https://git.mythiccraft.io/mythiccraft/MythicMobs/-/wikis/Skills/Mechanics/damage) mechanic

# Example
```yaml
Conditions:
- damageTag{tag=WITCHSCURSES}
```
The above will check damage for the tag WITCHCURSES. These tags can be anything and are used like the scoreboard tags feature.
