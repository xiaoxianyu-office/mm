## Description
Sets the caster's yaw and pitch to the same value of the target's


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| target    | to, t, location, loc, l, of | The [targeter](/Skills/Targeters)                  |<!--type:Targeter-->|


## Examples
```yaml
  Skills:
  - matchrotation @Parent
```
