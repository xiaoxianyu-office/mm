**Description:** Checks if the target entity has the given aura

---

**Attributes:**

| Attribute | Alias   | Description                       |
| --------- | ------- | --------------------------------- |
| auraName  | name, n | The name of the aura to check for |

---

**Examples:**

```
Conditions:
- hasaura{n=firedebuff} true
```

---

**Extra Information:**

- [x] Type: Entity
