**Description:** Checks if the target's Y location is above a value

---

**Attributes:**

| Attribute | Alias   | Description                   |
| --------- | ------- | ----------------------------- |
| height    | h       | The height they must be above |

---

**Examples:**

```
Conditions:
- heightabove{h=50} true
```

---

**Extra Information:**

- [x] Type: Location
