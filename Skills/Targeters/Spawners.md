## Description
Targets the locations of the specified mob spawners


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| spawners  | spawner, s| The names of the spawners                                            |         |


## Examples
```yaml
ExampleSkill:
  Skills:
  - effect:particles @Spawner{s=TestSpawner}
```
