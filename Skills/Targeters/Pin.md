## Description
Targets the position of a [Pin].


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| name      | n, pin, p, key, k | The name of the [pin] to target                              |         |


## Examples
```yaml
  Skills:
  - teleport @Pin{p=mypin}
```
