## Description
Checks whether the target player currently has a [Dialog](/Dialogs) open.

Dialogs are interactive UI windows opened with the [showDialog](/Skills/Mechanics/ShowDialog) mechanic. This condition is true from the moment a dialog is shown to the player until it is closed, either by the player or via the [closeDialog](/Skills/Mechanics/CloseDialog) mechanic.

> **This is a [Paper-Only] condition!**

> **This is a [Premium-Only] condition!**


## Attributes
*This condition has no attributes*


## Examples
```yaml
  Conditions:
  - hasDialogOpen true
```
