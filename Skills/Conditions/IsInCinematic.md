## Description
Checks if the target player is currently in an active cinematic camera path.

Returns true while the player is being moved by a [cinematicCamera](/Skills/Mechanics/CinematicCamera) mechanic, and false once the cinematic has ended or been cancelled.


## Attributes
*This condition has no attributes*


## Examples
```yaml
  Conditions:
  - isInCinematic true
```
